# Xiaomi Voice Checklist

## Browser Requirements
- Xiaomi Chrome 90+ or Mi Browser
- Android 10+
- Microphone permission granted

## Setup Steps
1. Open web app in Chrome on Xiaomi phone
2. When prompted, grant microphone permission
3. If not prompted: Settings > Apps > Chrome > Permissions > Microphone > Allow

## Voice Test Checklist
- [ ] Push-to-talk button visible
- [ ] Press and hold button → "Đang nghe..." appears
- [ ] Speak Vietnamese text → recognized correctly
- [ ] Release button → text sent as message
- [ ] Bot reply spoken aloud (TTS)
- [ ] Press PTT during TTS → speech interrupted

## Common Issues
| Issue | Fix |
|-------|-----|
| No mic permission | Settings > Apps > Chrome > Permissions > Microphone |
| STT not working | Ensure Chrome has internet connection |
| TTS not speaking | Check phone volume, try different voice in Chrome settings |
| Wrong language | Chrome > Settings > Languages > Add Vietnamese |
| PTT not responding | Try tap instead of hold, or reload page |

## Xiaomi-Specific Notes
- MIUI may restrict background mic access
- Battery saver mode may disable voice features
- Use Chrome, not Mi Browser, for best compatibility
